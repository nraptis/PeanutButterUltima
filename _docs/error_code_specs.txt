Error Code Spec (Simplified)

DEPRECATION DECREE
- Detailed integer code granularity is deprecated.
- Runtime priority is Safety and Fence behavior over exact legacy code selection.

Active grouped codes

`UNP_RHD_FENCE`
- Code name: Recovery header is invalid.
- Trigger variants:
- outside file fence
- inside recovery header fence
- inside archive header fence
- recover flow: outside selected-file fence

`UNP_FNL_FENCE`
- Code name: Path length is invalid.
- Trigger variants:
- `path_length == 0`
- `path_length > MAX_VALID_FILE_PATH_LENGTH`
- outside file fence
- inside recovery header fence
- inside archive header fence

`UNP_FDL_FENCE`
- Code name: File contents length is invalid.
- Trigger variants:
- `file_contents_length > MAX_VALID_FILE_CONTENT_LENGTH`
- outside file fence
- inside recovery header fence
- inside archive header fence

Other retained codes

`UNP_DEC_001`
- Trigger: decrypt failure during unpack/recover.

`UNP_EOF_001`
- Trigger: dangling archives after logical unpack stop.

`UNP_EOF_002`
- Trigger: dangling bytes after logical unpack stop.

`UNK_SYS_001`
- Trigger: unknown system failure.
