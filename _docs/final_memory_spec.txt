Archive Header (AH) (widened, memcpy-serialized):
struct ArchiveHeader {
  uint32_t mMagic;
  uint16_t mVersionMajor;
  uint16_t mVersionMinor;
  uint64_t mIdentifier;
  uint32_t mArchiveIndex;
  uint32_t mArchiveCount;
  uint32_t mPayloadLength;
  uint8_t mRecordCountMod256;
  uint8_t mFolderCountMod256;
  uint64_t mReserved;
};
total = 40 bytes

Field semantics:
- `mPayloadLength` is authoritative archive payload length for decode/discovery.
- `mRecordCountMod256` and `mFolderCountMod256` are hints only (modulo counters).

Write:
ArchiveHeader aHeader = ...;
memcpy(pDestination, &aHeader, sizeof(ArchiveHeader));

Recovery Header (RH):
struct RecoveryHeader {

Constants:
inline constexpr std::size_t SB_PAYLOAD_SIZE = 48; // fixed test target
inline constexpr std::size_t SB_L1_LENGTH = SB_PAYLOAD_SIZE + SB_RECOVERY_HEADER_LENGTH;
inline constexpr std::size_t SB_L2_LENGTH = SB_L1_LENGTH + SB_L1_LENGTH;
inline constexpr std::size_t SB_L3_LENGTH = SB_L2_LENGTH + SB_L2_LENGTH;

Archive length:
sizeof(ArchiveHeader) + archive_payload_length
where archive_payload_length is a positive multiple of SB_L3_LENGTH.

Record model (authoritative):
- There is NO explicit "file marker" byte.
- Record kind is inferred from the 6-byte content-length field.

Record wire format:
[2 bytes] path_length (little-endian uint16)
if path_length == 0:
  -> end-of-stream marker (no path bytes, no content-length field)
else:
  [path_length bytes] relative path
  [6 bytes] content_length_or_marker (little-endian uint48)
  if content_length_or_marker == 0xFFFFFFFFFFFF:
    -> directory record (no file-content bytes follow)
  else:
    -> file record, followed by exactly content_length bytes

Directory marker:
- Sentinel value: 0xFFFFFFFFFFFF (`kDirectoryRecordContentMarker`).
- This value is reserved for directory records only.

File-content length rules:
- File content length can be 0 (empty file is valid).
- No file-content padding is allowed inside the logical record stream.
- After end-of-stream, remaining payload bytes are trailing payload padding.

Payload shape:
[mixed file/directory records][single end-of-stream marker][trailing payload padding]
