EOF Consolidation Notes
=======================

Scope:
- Trailing-state code family was removed from decode.
- Legacy trailing-state codes were consolidated to `EOF_001`.
- `EOF_001` is surfaced as a primary error code through `OperationResult`.

Behavior:
- Unbundle:
  - if logical terminator is reached before readable payload ends, emit `EOF_001` terminally.
- Recover:
  - same condition emits `EOF_001` as recover-retry and continues invalid-walk logic.

Trailing-byte policy:
- No in-block post-terminator byte scan is part of the decode contract.
- The bundle-time tail zero-fill behavior remains intact.

Security-zone policy:
- Security zones are primary decode checks.
- One helper validates path-length, content-length, and recovery-stride spans.
- The helper applies:
  - readable-bytes bounds check
  - modulus-based "in-recovery-header" check
  - modulus-based "in-archive-header" check
- Security-zone violations emit one code: `SEC_ZONE_001`.
